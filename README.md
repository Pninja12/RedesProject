#### Como foi criado

Comecei por ter a ideia de tentar fazer a grelha do jogo do galo serem tiles numa grid do Unity, felizmente a ideia funcionou e assim permaneci até ao fim. O jogo decide um vencedor tendo o mapa verificando cada grelha quando um jogador usa uma peça,

#### Como funciona o online

O outline também foi criado com um shader graph que recebe onde o objeto está e como está no mundo e adiciona linha por detrás dele. A seguir ele é colocado em um renderer próprio que é atribuído a uma layer.
